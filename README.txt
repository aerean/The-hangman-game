The hangman game is written in C++ using OOP principles.
It has one class named joc, where there are informations about the game, such as:
-is game won?
-what is the solution?
-waht letters have been used?

It has a begining stage, where it tells you to choose a word which will be used in the game.
The second stage is the game, where you have to find out the letters forming the word.
The final stage is the won/lose screen, where it will ask if you want to play again.
